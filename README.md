# Full-Stack QR Ordering System

## Overview
The Full-Stack QR Ordering System is designed to streamline the dining experience by enabling quick, accurate table-side self-ordering. This system eliminates manual entry and enhances operational efficiency through real-time updates and a user-friendly interface.

## Features
- **Table-side Self-Ordering**: Customers can place orders directly from their smartphones using QR codes, reducing wait times and minimizing errors.
- **Real-Time Kitchen Display System (KDS)**: Updates are pushed to the kitchen staff in real-time via WebSocket, ensuring immediate order processing.
- **Manager Dashboard**: A comprehensive dashboard for managing orders, tables, and session controls. Managers can monitor and adjust orders seamlessly.
- **Mobile-First UI**: The user interface is designed with a mobile-first approach, ensuring optimal usability on smartphones and tablets.
- **Live Cart Functionality**: Customers can view and modify their cart in real-time, providing a dynamic ordering experience.

## Technologies Used
- **Frontend**:
  - React.js
  - WebSocket for real-time updates
  - HTML/CSS

- **Backend**:
  - Spring Boot
  - WebSocket for real-time communication
  - Database  MySQL
